# 图片处理工具

基于 React + Vite + Tailwind CSS 的在线图片处理应用，无需安装，打开即用。

## 功能

### 图片压缩
- 调整压缩画质（画质无损 ~ 高度压缩）
- 显示压缩前后文件大小对比
- 一键下载压缩后的图片

### 图片裁剪
- 拖动黄色框选择裁剪区域
- 实时预览裁剪效果
- 一键下载裁剪后的图片

### 图片适配
- 调整像素尺寸（宽度/高度）
- 保持比例缩放
- 显示物理尺寸信息
- 一键下载适配后的图片

### 其他功能
- 拖拽上传图片
- 显示图片详细信息（文件大小、像素尺寸、物理尺寸）
- 图片预览

## 技术栈

- React
- Vite
- Tailwind CSS
- Lucide React（图标）

## 使用

直接打开 `index.html` 即可使用，无需安装依赖。
